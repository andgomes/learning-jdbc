Persistindo Datas em Campos do Tipo Date

Neste post veremos como salvar campos do tipo Date em um banco de dados utilizando a classe Calendar do pacote java.util. Como exemplo, vamos criar uma tabela no banco de dados que salva datas(por motivos de simplicidade, ignore a utilidade dessa tabela).

CREATE TABLE datetable (
--some_date DATE
);

No código Java, se quisermos inserir um registro na tabela datetable, podemos utilizar um prepared statement e configurar o campo some_date passando como argumento um objeto do tipo java.sql.Date, obtido a partir do método getTimeInMillis() de um objeto Calendar.
